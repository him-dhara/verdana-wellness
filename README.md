## Brand System

| Role | Token | Hex |
|------|-------|-----|
| Light | Primary / Light | `#FBFFF5` |
| Dark | Primary / Dark | `#00545F` |
| Border | Primary / Border | `#00545F` at 10% |

**Typography:**

| Role | Font |
|------|------|
| Display / Headings | `Fraunces` - variable optical serif |
| Body / UI | `Bricolage Grotesque` - variable sans |

**Icons:** Phosphor - Duotone weight throughout.
